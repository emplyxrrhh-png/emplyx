The macOS-13 hosted image in Azure DevOps pipelines will be retired on December 4, 2025, and the Windows-2019 hosted image will retire on December 31, 2025 after a six-month extension. After these dates, any pipelines using these images will fail, so it is important to migrate to supported images in advance. For details on the brownout schedule and guidance on migration, please refer to the blog on Windows and Mac
