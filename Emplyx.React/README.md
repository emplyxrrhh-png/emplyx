# Emplyx.React

Nuevo frontend para Emplyx, desarrollado con React, TypeScript y Tailwind CSS.

## Requisitos

*   Node.js (v18 o superior)
*   npm

## Instalación

1.  Abre una terminal en esta carpeta:
    ```bash
    cd Emplyx.React
    ```
2.  Instala las dependencias:
    ```bash
    npm install
    ```

## Ejecución

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

## Estructura del Proyecto

*   `src/layouts/MainLayout.tsx`: Layout principal con Sidebar, Header y Footer.
*   `src/components/Sidebar.tsx`: Menú lateral colapsable con la estructura definida.
*   `src/components/Header.tsx`: Cabecera con perfil de usuario.
*   `src/App.tsx`: Configuración de rutas.

## Diseño

El diseño utiliza **Tailwind CSS** para un estilo fresco y moderno.
*   **Colores**: Indigo (Primary), Emerald (Secondary), Gray (Background).
*   **Iconos**: Lucide React.
